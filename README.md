#Full feature list:

Procedurally generated puzzles — every game is unique, using a Hamiltonian path algorithm
4 difficulty levels — 5×5 (Easy), 6×6 (Medium), 7×7 (Hard), 8×8 (Expert)
Click or drag to draw your path with smooth connector lines
Backtrack by dragging backwards or clicking the last cell
Gradient path line that shifts color as you progress
Hint system — highlights the next correct cell with a purple pulse
Undo / Clear controls for quick corrections
Timer tracking your solve speed
Win animation — cells flash green in sequence when solved
Puzzle counter — tracks total puzzles solved (saved in localStorage)
Keyboard shortcuts — Z to undo, R for new puzzle, H for hint
Mobile touch support — drag to draw path on touch devices
Dark neon aesthetic — blue/purple gradients, glowing path, amber number highlights
